KrokCom is a cross-platform application used to program the Krokodile
Cartridge from a serial port/USB-serial converter for Linux/UNIX and
maxOS systems.  Both GUI and commandline usage is supported.

If you'd like to verify that you have the latest release of KrokCom
visit the KrokCom Website at:

  https://github.com/sa666666/krokcom
